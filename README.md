# Перевірка методу Монте-Карло

## Опис завдання
Метою завдання було перевірити роботу методу Монте-Карло шляхом симуляції великої кількості кидків двох ігрових кубиків. Отримані емпіричні дані (частота випадіння кожної суми) були порівняні з теоретично розрахованими ймовірностями.

## Аналіз отриманих даних

1.  **Точність методу**: Результати показують надзвичайно високу точність симуляції. Абсолютна різниця між отриманими значеннями та теоретичними очікуваннями коливається в межах від **0.00%** до **0.05%**. Це свідчить про те, що вибірка була достатньо великою для отримання репрезентативних даних.

2.  **Закон великих чисел**: Експеримент наочно демонструє закон великих чисел: при збільшенні кількості випробувань середнє значення вибірки прагне до математичного сподівання. У нашому випадку емпіричні ймовірності майже збіглися з теоретичними.

3.  **Розподіл ймовірностей**:
    * Найменша похибка (**0.00%**) спостерігається для сум 3, 4 та 8.
    * Найбільша похибка (**0.05%**) зафіксована для суми 7. Це є очікуваним, оскільки сума 7 має найвищу ймовірність випадіння (найбільшу варіативність), проте навіть це відхилення є статистично незначущим у контексті даної задачі.

## Загальний висновок

Програмна реалізація алгоритму є **правильною**. Метод Монте-Карло успішно відтворив теоретичний розподіл ймовірностей сум двох кубиків.

Отримані результати підтверджують, що метод Монте-Карло є ефективним інструментом для наближеного обчислення ймовірностей у стохастичних процесах, де аналітичні розрахунки можуть бути складними або неможливими. Незначні відхилення є допустимими та пояснюються імовірнісною природою методу.
